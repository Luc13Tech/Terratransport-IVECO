Guide et Explications des Formules Excel

Voici le détail exact des calculs appliqués dans ce devis pour prouver une maîtrise parfaite du modèle financier sous Microsoft Excel.

1 Calculs Unitaires (1 Camion)

  • Coût Jour : = 250.000 FCFA

    C'est la valeur de base fixée par le partenariat.

  • Coût Semaine (6 jours max) : = 250.000 × 6 = 1.500.000 FCFA

    Excel: =Tarif_Jour * 6

  • Coût Mois (25 jours max) : = 250.000 × 25 = 6.250.000 FCFA

    Excel: =Tarif_Jour * 25

  • Coût An (12 mois) : = 6.250.000 × 12 = 75.000.000 FCFA

    Excel: =Tarif_Mois * 12

2 Calculs Global Flotte (100 Camions)

  • Flotte Par Jour : = 100 × 250.000 = 25.000.000 FCFA

    Excel: =C10 * $B$4 (Référence absolue)

  • Flotte Par Semaine : = 100 × 1.500.000 = 150.000.000 FCFA

    Excel: =C11 * $B$4

  • Flotte Par Mois : = 100 × 6.250.000 = 625.000.000 FCFA

    Excel: =C12 * $B$4 (625 millions FCFA/mois)

  • Flotte Par An : = 100 × 75.000.000 = 7.500.000.000 FCFA

    Excel: =C13 * $B$4 (7,5 Milliards FCFA HT)

Astuce de Pro Excel : Format de Nombre Personnalisé

Dans Excel, pour afficher automatiquement la monnaie sans casser les calculs numériques, appliquez le format de cellule suivant : #,##0 "FCFA"
